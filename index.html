Custom Development

Production Systems

Enterprise-grade systems built from scratch using Laravel, React, and WhatsApp Business API — currently powering dealership operations across South Africa.

01 — LIVE SYSTEM

Lead2Call

Automated lead-to-call platform for automotive dealerships. When a customer submits a vehicle enquiry online, the system immediately initiates an outbound call connecting the customer to the next available salesperson using a round-robin queue — no manual intervention required.

"Salespeople are cycled through in order, and if a salesperson misses the call within a defined window, the system moves to the next person in the rotation. When a call is missed, the salesperson receives a WhatsApp notification presenting the lead details in a structured message window."

Laravel 12 React Inertia.js Sudonum API WhatsApp Business API MySQL
02 — LIVE SYSTEM

WhatLead

WhatsApp-powered lead management platform built for automotive dealerships. When a customer submits a vehicle enquiry, WhatLead's API triggers an automated messaging workflow entirely over WhatsApp Business. The customer receives an immediate branded response, including the dealership's catalog and the assigned salesperson's contact vCard.

"The platform supports multiple dealerships, each with their own WhatsApp Business number(s), salesperson roster, service configuration, and automation rules. Includes a full conversation dashboard, two-layer template management system (designs submitted to Meta for approval), and vehicle inventory sync with trackable short links."

Laravel 12 React Inertia.js v2 WhatsApp Business API Meta Cloud API REST API
